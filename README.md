Dungeon.gmx
===========

This is a rogue-like RPG game with randomly generated dungeons. 

Features
========

Working
-------
* Randomly generated dungeon
* Collision detection
* Basic melee combat
* Awful UI
* Scrolling combat log

In-progress
-----------
* Better UI
* More monsters
* Ranged combat
* Hero levels
* Pick ups (potions, weapons, armor)
